Онлайн чат, который случайно соединяет людей.

Функциональные требования:

1.Пользователи могут зарегистрироваться в системе, указав свои уникальные идентификаторы (имя пользователя и пароль)
2.Пользователи могут отправлять и принимать текстовые сообщения друг от друга
3.Система случайным образом соединяет пользователей между собой, чтобы они могли общаться
4.Пользователи могут покинуть чат в любой момент

Нефункциональные требования:
1.Система должна быть реализована на языке программирования Python, с использованием веб-фреймворка Flask
2.Для хранения данных используется база данных SQLalchemy
3.Для аутентификации пользователей используется JWT
4.Разработка должна быть выполнена с использованием модульного и инкрементального подхода.

Техническое задание:

1.Разработать и протестировать API, которое будет использоваться для общения с клиентской стороной. API должен предоставлять методы для регистрации, аутентификации.
2.Разработать и протестировать базу данных SQLalchemy для хранения данных пользователей и сообщений.
3.Разработать и протестировать серверное приложение на языке Python с использованием Flask, которое будет обрабатывать запросы от клиентской стороны, взаимодействовать с базой данных и осуществлять соединение между пользователями.
4.Разработать и протестировать клиентскую сторону приложения с использованием HTML, CSS, JavaScript.
5.Продемонстрировать работу приложения
